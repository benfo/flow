## Linear

[Linear](https://linear.app) is the first supported provider.

### Authentication

Run `flow` for the first time and follow the authentication prompt. flow uses the Linear OAuth flow and stores the token locally.

### Configuration

flow reads your Linear workspace automatically once authenticated. No additional configuration is required to get started.

## Planned providers

The following providers are on the roadmap:

- **GitHub Issues**
- **Jira**

If you would like to contribute a provider, see the [GitHub repository](https://github.com/benfo/flow).
