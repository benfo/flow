## Global

| Key | Action |
|-----|--------|
| `?` | Toggle help overlay |
| `q` / `Ctrl+C` | Quit |

## Task Detail

| Key | Action |
|-----|--------|
| `â†‘` / `â†“` | Scroll content |
| `Tab` | Toggle subtask list focus |
| `Enter` | Open selected subtask |
| `p` | Navigate to parent task |
| `o` | Open task URL in browser |
| `y` | Copy task URL (or ID) to clipboard |
| `b` | Create or checkout git branch |
| `P` | Open pull request in browser (active branch only) |
| `e` | Edit task |
| `n` | Create subtask |
| `s` | Change status |
| `a` | Assign to self |
| `c` | View / manage comments |
| `D` | Delete task (prompts confirmation) |
| `Esc` / `Backspace` | Go back |

## Comments

| Key | Action |
|-----|--------|
| `â†‘` / `â†“` | Navigate comments |
| `n` | Add a new comment |

## Comment Compose

| Key | Action |
|-----|--------|
| `Ctrl+S` | Save comment |
| `Esc` | Cancel (prompts if there are unsaved changes) |

## Create / Edit Task

| Key | Action |
|-----|--------|
| `Tab` / `Shift+Tab` | Next / previous field |
| `Ctrl+S` | Save |
| `Esc` | Cancel (prompts if there are unsaved changes) |
